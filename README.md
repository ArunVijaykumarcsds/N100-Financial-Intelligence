# N100 Financial Intelligence Platform

## Sprint 1 - Data Foundation

### Objective

Build a fully validated SQLite database (nifty100.db) from 12 source Excel files.

### Deliverables

* nifty100.db
* loader.py
* validator.py
* normaliser.py
* schema.sql
* load_audit.csv
* validation_failures.csv
* 35+ unit tests

### Technology Stack

* Python
* Pandas
* NumPy
* SQLite
* SQLAlchemy
* Pytest
* Streamlit
* FastAPI

### Sprint Goal

Create a production-ready ETL pipeline that loads, validates, normalizes and stores Nifty100 company financial data.
